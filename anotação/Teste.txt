Desafio: Criar uma loja do tipo que você quiser (Exemplo: Petshop, Venda de itens de personagens, Itens ficticios)


SUGESTÃO DE APIs abaixo (Obs: algumas são mais simples, outras mais completas, foquem em consumir o conteudo que você deseja):

https://thedogapi.com/ - API de cachorros
https://deckofcardsapi.com/ - API de cartas de baralho
https://pokeapi.co/ - API de pokemon
https://rickandmortyapi.com/ - API Rick and Morty
https://developer.marvel.com/?ref=devresourc.es - API da Marvel
https://www.scorebat.com/video-api/?ref=devresourc.es - API de futebol (eu acho)

Estrutura básica da loja (configurações do HTML)

Nome da Loja

Body:
